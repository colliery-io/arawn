# Tutorials

Hands-on guides that walk you through real tasks from start to finish. Each
tutorial is designed to be followed step-by-step, with concrete commands and
expected output at every stage.

| Tutorial | Description |
|----------|-------------|
| [Getting Started with Arawn](getting-started.md) | Install Arawn, run your first query, explore the chat REPL and TUI, and understand workstreams. |
| [Setting Up Memory and Knowledge](memory-setup.md) | Configure persistent memory, store and recall facts, and explore the knowledge graph. |
| [Building Your First Workflow](first-workflow.md) | Enable the pipeline engine, write a workflow in TOML, use WASM runtimes, and schedule with cron. |
| [Creating Your First Plugin](first-plugin.md) | Build a plugin with a skill, a hook, and a custom agent, then install and test it. |
