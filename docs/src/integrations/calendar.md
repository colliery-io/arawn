# Google Calendar

Lets the agent answer "what's on my calendar tomorrow," schedule new events, and check for conflicts before suggesting times. Three tools land when Calendar is configured: `calendar_upcoming`, `calendar_create_event`, `calendar_find_conflicts`.

For setup, see the [Connecting integrations](../getting-started.md#6-connect-integrations) section of Getting Started — Calendar is covered under the **Google** subsection alongside Gmail and Drive (one Google Cloud project covers all three).

After `/connect google_calendar` succeeds, try:

```
what's on my calendar this week?
```
