# Gmail

The Gmail integration lets the agent read your inbox, search messages, send mail, and mark messages read. Five tools land when configured: `gmail_inbox_read`, `gmail_search`, `gmail_get_message`, `gmail_send`, `gmail_mark_read`.

For setup, see the [Connecting integrations](../getting-started.md#6-connect-integrations) section of Getting Started — Gmail is covered under the **Google** subsection alongside Calendar and Drive (one Google Cloud project covers all three).

After `/connect gmail` succeeds, try:

```
list the 5 most recent unread emails in my inbox
```
